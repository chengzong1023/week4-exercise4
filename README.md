## Description
This repository contains the completed Week 4 student worksheet for the geospatial analysis course.

## Week 4 Topics
1. Vector aggregation (dissolve & groupby)
2. Raster data as NumPy arrays with spatial metadata
3. Terrain slope computation from DEM
4. Extract raster values into vector shapes (zonal statistics)

## Requirements
- Python 3.11+
- geopandas
- rioxarray
- rasterstats
- matplotlib
- folium
- mapclassify

## Usage
```bash
# Clone the repository
git clone https://github.com/chengzong1023/week4-exercise4.git
cd week4-exercise4

# Install dependencies
pip install geopandas rioxarray rasterstats matplotlib folium mapclassify

# Open the notebook
jupyter notebook Week4-Student.ipynb
```
